# ghizlan-charity-system

#  نظام جمعية غزلان الخير الإنسانية
> **نظام متكامل لإدارة طلبات المساعدة وتوثيق الحالات الإنسانية.**

---

##  نظرة عامة
يستهدف المشروع تحويل العمل الإداري في الجمعية من النظام التقليدي (إكسل وواتساب) إلى نظام رقمي مؤتمت بالكامل، يوفر واجهة سهلة للمحتاجين ولوحة تحكم ذكية للموظفين لاتخاذ قرارات دقيقة وسريعة.

---

## 🛠 التقنيات المستخدمة (Stack)
* **Backend:** Laravel (PHP 8+)
* **Frontend:** Next.js (React)
* **Database:** MySQL
* **File Storage:** Cloudflare R2
* **Infrastructure:** VPS + SSL + Cloudflare Security

---

## 🚀 الميزات الرئيسية
* ✅ **تقديم طلبات مجهولة:** للمحتاجين دون الحاجة لتسجيل حساب.
* ✅ **نظام إشعارات:** تنبيهات تلقائية عبر (WhatsApp, SMS, Email).
* ✅ **فلترة ذكية:** تصنيف الحالات حسب المنطقة، النوع، والأولوية.
* ✅ **لوحة قياس أداء (KPIs):** مراقبة سرعة معالجة الطلبات ونسب القبول.
* ✅ **ثنائي اللغة:** دعم كامل للعربية والإنجليزية.

---

##  الأمان وخصوصية البيانات
* تشفير البيانات الحساسة (الطبية والمالية).
* نظام صلاحيات دقيق (Admin, Employee).
* حماية ضد الهجمات عبر Cloudflare.
